Störungsbeseitigung in einem Silo
Fallbeispiel II
In einem Silo hat sich Natron zu einer Brücke verklumpt, so dass kein Füllgut mehr gefördert wird. Die Natron-Brücke kann nur von innen beseitigt werden.
Welche Gefährdungen können auftreten, wer ist beteiligt und welche Schutz­maßnahmen müssen getroffen werden, damit das Silo sicher befahren werden kann? Helfen Sie dem Unternehmer bei seinen Aufgaben.
