Hinweis für den IE: Anzeigen aktiver Inhalte
 

Das Anzeigen aktiver Inhalte
Wenn das Informationssystem von einer lokalen Festplatte oder einer CD aufgerufen wird, kann es vorkommen, dass der Internet Explorer folgende Fehlermeldung ausgibt:

„Das Anzeigen aktiver Inhalte, die auf den Computer zugreifen können, wurde für diese Datei aus Sicherheitsgründen eingeschränkt. Klicken Sie hier, um Optionen anzuzeigen…”

Damit diese Fehlermeldung nicht mehr angezeigt wird, gehen Sie bitte wie folgt vor: Öffnen Sie die Einstellungen für den Internet Explorer (Extras → Internetoptionen…) und wählen den Karteireiter „Erweitert” aus.

Scrollen Sie die Einstellungen nach unten bis Sie zu dem Bereich „Sicherheit” gelangen. Setzen Sie die Checkmarken bei folgenden Optionen:

  • Ausführung aktiver Inhalte in Dateien auf dem lokalen Computer zulassen
  • Ausführung aktiver Inhalte von CDs auf dem lokalen Computer zulassen
